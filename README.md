# VN-index-sector-analysis
This project analyses sector-level return dynamics across Vietnam's HOSE equity market from 2019 to 2025, covering Banking, Real Estate, Consumer Staples, and Information Technology. The paper examines what drives sector behaviour and what it implies for portfolio construction.

# Data
- **Stocks:** VCB, STB (Banking), VHM, NVL (Real Estate), VNM, MSN (Consumer Staples), FPT (Information Technology)
- **Period:** 2019–2025
- **Source:** Investing.com / HOSE

# Methods
- OLS regression (beta, alpha, R²)
- Rolling correlation analysis (180-day window)
- Rolling volatility analysis (90-day window)

# Code
The analysis is in `VN_Index_Sector_Analysis.ipynb`, run in Google Colab.
